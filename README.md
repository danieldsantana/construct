# 🏗️ CONSTRUCT
### Sistema de Controle de Recebimento de Materiais

Sistema web desenvolvido em Python com Streamlit e PostgreSQL para controle de recebimento de materiais em obras de transmissão de energia elétrica.

## Funcionalidades

- Cadastro de notas fiscais e romaneios de recebimento
- Controle de estoque recebido vs previsto por trecho de obra
- Registro de pendências e não conformidades
- Exportação de relatórios
- Gestão de usuários com perfis de acesso (Administrador, Operador, Visualizador)

## Stack

- **Frontend:** Streamlit
- **Backend:** Python + SQLAlchemy
- **Banco de dados:** PostgreSQL (Supabase)
